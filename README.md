PIMO-IDE: Projection-Iterative-Methods-Based Single Image Detail Enhancement

This repository provides the official MATLAB implementation of our paper on **PIMO-based single image detail enhancement**.

Project Introduction

This project presents an image detail enhancement framework based on a **Projection-Iterative-Methods-Based Optimizer (PIMO)**.  
The method models local patch matching as a population-based optimization problem and combines multi-scale decomposition, gradient/texture feature constraints, and PIMO search operators to enhance image details while preserving structural consistency.

Given an input image, the algorithm first constructs a smooth reference layer through an upsampling–downsampling strategy and obtains an initial high-frequency residual layer. It then performs local patch retrieval under joint intensity, gradient, and texture constraints, and reconstructs a more reliable detail layer through PIMO-based optimization. Finally, the enhanced result is generated by adding the recovered detail layer back to the original image.

Compared with conventional greedy or purely filter-based strategies, the proposed method is designed to provide:
- stronger local structure preservation,
- more reliable texture restoration,
- better resistance to pseudo-texture introduction,
- and improved matching stability via projection-guided optimization.

Requirements

MATLAB R2024a / R2024b recommended
Image Processing Toolbox
Parallel Computing Toolbox (optional but recommended for the `parfor` version)


Quick Start

Step 1: Prepare the environment
Make sure MATLAB is installed correctly and the required toolboxes are available.

Step 2: Prepare the files
Place the following files in the same working directory:
`DE_demo.m`
`PIMO_parfor.m`
`grad.m`
`texture.m`

Create the following folders if they do not already exist:
`data`
`results`

Step 3: Place input images
Put the input images to be enhanced into the `data` folder.

Step 4: Run the program
Open MATLAB, set the **Current Folder** to the project directory, and run:

matlab
DE_demo


or click the Run button in the MATLAB Editor.


Notes

1. First-run speed
If the parallel version is used, MATLAB may start a parallel pool (`parpool`) during the first run.  
This initialization may take additional time, which is normal.

2. Memory usage
For high-resolution images, the padding, local search, and patch-based accumulation may consume a considerable amount of memory.

3. Parallel acceleration
`PIMO_parfor.m` uses `parfor` to accelerate patch-level processing.  
The actual speedup depends on:
-CPU core count,
-MATLAB parallel configuration,
-image size,
-and the selected parameter settings.
 4. File naming
Please keep the helper functions (`grad.m`, `texture.m`) and the main algorithm file in the same directory as `DE_demo.m`.

Method Overview

The current implementation mainly includes the following stages:

1. Multi-scale decomposition
   Construct a reference layer and initial high-frequency residual.

2. Feature extraction**
   Extract gradient features using the Sobel operator and texture features using the Laplacian operator.

3. Local patch matching 
   Search for optimal candidate patches in a local neighborhood.

4. PIMO-based optimization  
   Use population evolution with multiple operators to improve matching stability and avoid poor local solutions.

5. Detail reconstruction  
   Recover a reliable detail layer and add it back to the original image for enhancement.
